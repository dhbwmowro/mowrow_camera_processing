# Depth Detection


The code was built and run with ROS melodic and Ubuntu 18.04.

## Dependencies
   * Ubuntu 18.04
   * ROS Melodic
   * OpenCV
   * Intel librealsnese2
   * Intel Realsense D435
   
## Publishes:

* `/uav1/camera/depth` (sensor_msgs::Image)


## Running the code

* `roslaunch depth_detection depth_detection.launch`
